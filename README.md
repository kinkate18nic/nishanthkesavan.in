# nishanthkesavan.in — Personal Portfolio

Personal portfolio and CV website for Nishanth Kesavan.

- 🏠 **Home** — Hero landing page with featured projects
- 👤 **About** — Full CV with work experience, skills, education
- 📂 **Projects** — Portfolio showcase of all projects
  - 🚂 [RailHygiene](/projects/railhygiene/) — Android train hygiene rating app
  - 🌊 [Kerala Flash-Flood Watch](/projects/flash-flood-keralam/) — Monsoon monitoring dashboard
  - 🧬 [ProteinDB](/projects/protein-aggregator/) — Indian protein supplement scorer
  - 📖 [Padham](/projects/padham/) — Malayalam spiritual text reader

## Tech

Static HTML/CSS/JS. No framework. GitHub Pages + custom domain.

## Deploy

Push to `main` branch → GitHub Pages auto-deploys.

Custom domain: `nishanthkesavan.in` (configured via CNAME file).

## Local development

```bash
npx http-server . -p 8080 -c-1
```
